# Indian_house_prices_ prediction

## 📌 Overview

This project predicts house prices in India using Machine Learning.
It also classifies whether a property is a **good investment** based on price per square foot.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Outlier Handling using IQR
* Data Visualization (Heatmap)
* Linear Regression for Price Prediction
* Logistic Regression for Investment Classification
* Model Saving using Pickle

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

---

## ⚙️ Workflow

1. Load dataset
2. Handle missing values & outliers
3. Encode categorical features
4. Train models:

   * Linear Regression (price prediction)
   * Logistic Regression (investment classification)
5. Evaluate performance
6. Save models

---

## ▶️ How to Run

```
pip install -r requirements.txt
python indian_house_prices.py
```
